# Task Manager

Fullstack-приложение для управления задачами.

## Структура репозитория

| Папка | Назначение |
|-------|------------|
| **frontend/** | Клиент на React (сборка Vite) |
| **backend/** | Сервер на Python FastAPI |

## Локальный запуск (после установки зависимостей)

1. **Frontend:** в каталоге `frontend/` выполните `npm install`, затем `npm run dev`.
2. **Backend:** настройте виртуальное окружение в `backend/`, установите зависимости FastAPI и запустите приложение (инструкции появятся на следующих этапах).

## Требования

- Node.js (для frontend)
- Python 3.x (для backend)
